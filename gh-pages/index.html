youtube-embedr

Default implementation

This is the stock standard implementation. Has a ratio of 16:9 and a width of 560px

<div class="youtube" id="5GedjRxj9EA"></div>

Custom width/height

Set the width or height, and it'll work out the ratio for you.

<div class="youtube" id="GOtxJrzp6ls" style="width:370px;"></div>

Fluid

Fluidity will make it fill 100% of the parent width.

<div class="youtube" id="R4ajQ-foj2Q" style="width:100%;" data-fluid="true"></div>

Default title

Want a title? (Looks like the API was updated and this is pretty average now)

<div class="youtube" id="R4ajQ-foj2Q" style="width:100%;" data-fluid="true" data-title></div>

Custom title

Want a custom title?!

<div class="youtube" id="R4ajQ-foj2Q" style="width:100%;" data-fluid="true" data-title="I'm custom..."></div>

Lightbox

Want video to open in a lightbox?

Requires ekko-lightbox jQuery plugin.

<div class="youtube" id="R4ajQ-foj2Q" style="width:100%;" data-fluid="true" data-toggle="lightbox"></div>
